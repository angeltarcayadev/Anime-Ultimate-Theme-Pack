# Deku Series: Ultimate Dark Theme 🦸‍♂️🔥

¡Lleva tu VS Code al siguiente nivel con el tema definitivo inspirado en Deku! Este no es un solo un tema de colores, es una experiencia visual completa que incluye fondos de pantalla personalizados y transparencias tipo cristal.

## ✨ Características Especiales
- **Wallpaper de Deku Integrado**: Fondo de pantalla de alta calidad.
- **Transparencia "Glass Mode"**: Interfaz traslúcida que permite ver el fondo sin perder legibilidad.
- **Auto-Update**: Cambia la configuración en tiempo real y mira los resultados al instante.
- **Configuración Personalizada**: Cambia la opacidad, el tamaño o pon tu propia imagen desde los ajustes de VS Code.

## 🚀 Instalación
1. Descarga el archivo `.vsix` de la sección de [Releases](tu-link-a-futuro).
2. En VS Code, ve a la pestaña de Extensiones, haz clic en los tres puntos `...` y elige **Install from VSIX**.
3. Presiona `Ctrl + Shift + P` y ejecuta el comando: `Deku Theme: Instalar Wallpaper`.
4. Reinicia VS Code.

## 🛠️ Personalización
Ve a `Settings` (Ctrl + ,) y busca **Deku Theme**:
- **Opacity**: Ajusta la intensidad del fondo.
- **Path**: Pon la ruta de tu propia imagen favorita.
- **Size**: Elige entre `cover`, `contain` o `auto`.

## ⚠️ Nota de Compatibilidad
Al modificar el CSS interno de VS Code para permitir transparencias, podrías ver un aviso de **"Installation is corrupt"**. Esto es **NORMAL** y esperado al inyectar estilos personalizados. Simplemente haz clic en el engranaje del aviso y selecciona "Don't show again".
